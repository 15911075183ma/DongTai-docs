# 洞态 Server

## 概述

本次测试是对洞态 IAST 的性能验证和稳定性验证，确保产品的稳定运行。

## 测试对象

* 产品 : 洞态 IAST
* 版本 : 1.4.0
* 接口 : Agent 数据上传接口

## 测试目标

* 确保产品主要接口稳定运行，无内存泄漏
* 确保产品在高并发下长时间稳定性运行
