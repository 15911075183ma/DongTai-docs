# 组件漏洞

## 查询 JAVA 组件漏洞

* **API Path**： 

	`iast.io/openapi/sca/v1/package_vul/?hash=07b6bf82cea13570b5290d6ed841283a1fcce170`


* **请求方法**：
`GET`

* **详细说明**：
通过组件 `SHA-1` 信息获取组件对应的漏洞数据

* **REST 参数**：

	|参数名|是否必填|类型|
	|:|:|:|
	|hash |必填 |string |

* **请求内容**：

	```
	/openapi/sca/v1/package_vul/?hash=cfa4f316351a91bfd95cb0644c6a2c95f52db1fc
	```

* **字段解释**：

	|:|:|
