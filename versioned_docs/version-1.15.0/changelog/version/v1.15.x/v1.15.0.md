# v1.15.0

发布日期：2023 年 09 月 15 日

## **产品能力**

* 平台UI架构优化二期
* 拓扑图运算框架及交互逻辑调整
* 漏洞原始调用链的转存和下载

## **检测能力**

* 敏感信息检测常用规则内置
* 优化检测引擎的内存消耗
* 重构了IastClassDiagram类，去除了不必要的锁，避免多Agent时的死锁问题
* 新增110个hook点
* junit和jmh提升到全局级别，增加部分单元测试
* 支持org.apache.httpcomponents:fluent-hc全版本添加traceId
* 移除API收集时的部分无用日志 
* 上报时的消息性能优化
* 上报线程池参数可配置
* 可根据漏洞类型配置自定义污点 tag
* 优化长调用链下内存和cpu占用
